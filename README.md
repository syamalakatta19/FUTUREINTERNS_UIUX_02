# 🌿 Plant Information Mobile App UI Design

## 📖 Project Overview

The Plant Information Mobile App is a mobile UI/UX design project created using Figma. The application helps users explore different categories of plants, learn about their characteristics, and book appointments for plant-related guidance and care.

The design focuses on providing an attractive, nature-inspired interface with easy navigation and informative plant content.

---

## 🎯 Objectives

* Provide information about different types of plants.
* Create an engaging and user-friendly mobile interface.
* Help users identify plant categories easily.
* Allow users to book appointments for plant consultation.
* Promote awareness about indoor and outdoor plants.

---

## 📱 Screens Included
